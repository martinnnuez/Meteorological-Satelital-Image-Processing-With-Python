Práctico de transferencia radiativa (Modelo Directo)¶

       Elaborado por Franco Ompré 2019
       Revisado por Sebastián Heredia 2020

El objetivo del práctico es poder setear un conjunto de variables que nos permitan calcular la radiación a tope de atmósfera para un dado escenario utilizando el Modelo de Transferencia Radiativa 6S.

Para conocer la radiación a TOA (Top Of Atmosphere) necesitamos fijar un conjunto de parámetros que caractericen la escena observada.

El 6S es un código escrito en lenguaje Fortran 77 por lo que vamos a utilizar una libreria de python llamada Py6s que nos sirve de interfaz con el 6S permitiendonos acceder a la configuración del modelo de transferencia radiativa de forma mas simple.

Necesitamos configurar el siguiente conjunto de parámetros:

- Geometria de observación:
    - Altitud de observador
    - Altitud del blanco
    - Ángulos de observación del blanco.
    - Angulos de iluminación del blanco.

- Condiciones Atmosféricas:
    - AOT (Aerosol optical thickness)
    - Aero Profile
    - Atmospheric profile
    - Atmospheric correction

- Superficie

Luego de tener seteados los parametros para la simulación resta correr el Py6s. Al utilizar el comando run el Py6s genera un archivo de texto, el cual, es el input del 6S. Al ejecutarse el 6S, su output es un archivo de texto que contiene las variables de salida (ej: radiacia del pixel a TOA), las cuales son importadas e interpretado por el Py6s (para más información, consultar: RWilson_20200508_Py6S.pdf)

A continuación se mostrarán diferentes configuraciones para ejecutar el 6S via el Py6S.

Pregunta 1: ¿A qué ángulo aproximadamente se encuentra el máximo de radiancia? ¿Porque? ---- A aproximadamente 78 grados se observa el maximo de irradiancia, esto es asi porque a mayores angulos cenitales, menor es el camino que recorre la radiacion solar en la atmosfera por lo que se reduce la capacidad de la misma para atenuarlo describiendo el maximo de irradiancia. ----

Cuanto más perpendicular se encuentra el sol con respecto a la superficie terrestre (es decir, cuanto menor valor del ángulo cenital) menor es el camino que recorre la radiación solar a través de la atmósfera.

Por el contrario para ángulos cenitales mayores (menor altura solar) el camino a recorrer por la radiación solar en la atmósfera es mayor, lo que implica que la intensidad de la radiación solar que llega a la superficie terrestre es menor.
